## 服务通信协议

### 1. 前端 ↔ 后端服务
```
协议: HTTP/HTTPS
格式: JSON
认证: JWT Token (Header: Authorization: Bearer <token>)
```

### 2. Cybertwin-Agent ↔ 专科医生服务
```
协议: HTTP
格式: JSON
认证: API Key (Header: X-API-Key: <key>)
```

### 3. 服务 ↔ 数据库
```
协议: MySQL Protocol
连接池: aiomysql (异步) / pymysql (同步)
数据库: MySQL 8.0（云数据库）
地址: 192.168.208.66:31717
字符集: utf8mb4
```
